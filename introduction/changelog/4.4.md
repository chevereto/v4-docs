# 4.4 Vivaracho

*Vivaracho is currently under development.*

Chevereto v4.4, codenamed [**Vivaracho**](https://dle.rae.es/vivaracho), adds first-class multi-tenancy, enabling you to provision, isolate, and administer multiple independent sites from a single installation.

## Environment changes

The following environment variables have been added or modified:

`CHEVERETO_ENABLE_TENANTS`

This variable enables or disables multi-tenancy support in Chevereto. When set to `1`, administrators can create and manage multiple independent sites within a single Chevereto installation.
